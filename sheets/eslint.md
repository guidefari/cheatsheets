## Wes Bos ES-Lint config

To install locally/per project:

`npx install-peerdeps --dev eslint-config-wesbos`

and create a `.eslintrc` file:

```json
{
  "extends": ["wesbos"]
}
```

full details on the config
[here](https://github.com/wesbos/eslint-config-wesbos).
